# vite-test-for-me

this is `react` with `vite`( not webpack ), it just learn for `react-router`(v6) and `recoil`(0.7)

这个是用vite搭建的react，只是为了学习react-router(v6)和recoil(0.7)相关内容的

### develop 开发

install 下载依赖
```shell
npm install
```
running 项目运行
```shell
npm run dev
```

### product 生成
build project 项目打包
```shell
npm run build
```
